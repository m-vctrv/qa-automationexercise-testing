## REG-005 — Валидация текстовых полей (Name, First name, Last name, State, City)

| Параметр | Значение |
|---|---|
| Приоритет | Средний |
| Тип | Негативный (data-driven) |
| Предусловия | Открыта форма регистрации |
| Правило | Допустимы буквы, дефис, точка, пробел внутри. Длина 1–50 |

**Общие шаги:** ввести значение из таблицы в проверяемое текстовое поле → отправить форму.

| # | Входные данные | Ожидаемый результат |
|---|---|---|
| 1 | (пусто) — для обязательного поля | Не проходит; поле обязательно |
| 2 | `   ` (только пробелы) | Не проходит |
| 3 | `Иван123` (цифры) | Не проходит; цифры недопустимы |
| 4 | `Иван!@#` (спецсимволы) | Не проходит; спецсимволы недопустимы |
| 5 | строка из 51 символа (граница) | Не проходит; превышена длина (макс. 50) |
| 6 | `A` (1 символ, граница) | Проходит |
| 7 | `Anna-Maria` (буквы + дефис) | Проходит |

## REG-007 — Валидация поля Mobile Number

| Параметр | Значение |
|---|---|
| Приоритет | Средний |
| Тип | Негативный (data-driven) |
| Предусловия | Открыта форма регистрации |
| Правило | Только цифры (допускается ведущий `+`). Длина 11 без `+` / 12 с `+` |

**Общие шаги:** ввести значение из таблицы в Mobile Number → отправить форму.

| # | Входные данные (Mobile) | Ожидаемый результат |
|---|---|---|
| 1 | (пусто) | Не проходит; поле обязательно |
| 2 | `abcdefghijk` (буквы) | Не проходит; только цифры |
| 3 | `7123!@#0000` (спецсимволы) | Не проходит |
| 4 | `7123456789` (10 цифр, граница) | Не проходит; слишком короткий |
| 5 | `712345678901` (13 символов, граница) | Не проходит; слишком длинный |
| 6 | `71234567890` (11 цифр, валидный) | Проходит |
| 7 | `+71234567890` (12 с `+`, валидный) | Проходит |
