# BUG-001 — Нарушена сетка карточек товаров в каталоге

| Параметр | Значение |
|---|---|
| ID | BUG-001 |
| Модуль | Каталог |
| Среда | Яндекс Браузер 26.4.5.634 (64-bit), Windows 10 22H2 |
| Стенд | prod (https://automationexercise.com/) |
| Severity | Minor |
| Priority | Medium |
| Статус | Open |
| Автор | Викторов М.С. |
| Дата | 24.06.2026 |

## Описание

Карточки товаров в каталоге имеют разную высоту, из-за чего нарушается сетка: в рядах появляются пустые места, товары располагаются непоследовательно.

## Шаги воспроизведения

1. Открыть https://automationexercise.com/
2. В меню слева выбрать категорию **BRAND → BIBA**
3. Прокрутить к списку товаров
4. Осмотреть ряды карточек

## Фактический результат

Карточки товаров в ряду имеют разную высоту, сетка ломается — появляются пустые ячейки, товары идут непоследовательно.

## Ожидаемый результат

Все карточки товаров имеют одинаковую высоту, сетка ровная, товары располагаются последовательно без пустот.
    
## Дополнительная информация

Проверено в DevTools: высота карточек различается — **490.6px** против **411.8px** (ширина одинаковая — 207.5px). Причина разъезда сетки — разная высота карточек из-за разной высоты изображений/контента.
